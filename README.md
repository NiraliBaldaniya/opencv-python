# Face Detection using OpenCV

- Detects faces with a blue rectangle.
- Displays **"eye detected"** when two eyes are found.
- Displays **"smile detected"** when a smile is detected.

## Requirements

Install OpenCV:

```bash
pip install opencv-python
```

## Files

```
face-eye-smile-detection.py
haarcascade_frontalface_default.xml
haarcascade_eye.xml
haarcascade_smile.xml
```

## How to Run

```bash
python face_detection.py
```

Press **Q** to quit.
